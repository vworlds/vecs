# vecs monorepo

This repository contains the `@vworlds/vecs` TypeScript libraries.

## Packages

- `lib/vecs`: the existing `@vworlds/vecs` ECS package.
- `lib/vecs-client`: placeholder for browser-oriented vecs code.
- `lib/vecs-server`: placeholder for Node.js-oriented vecs code.

## Development

```sh
npm install
npm run lint
npm run typecheck
npm run test
npm run build
```
